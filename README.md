# Stone Paper Scissor Game

A simple command-line Stone Paper Scissor game built using Python.

## Features
- 🎮 Play against the computer
- 🎲 Random computer choice
- 📊 Scoreboard
- ✅ Input validation
- 🔄 Play again option

## Technologies Used
- Python 3
- Random Module

## How to Run

```bash
python game.py
```
